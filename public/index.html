QueryGate

Read-only PostgreSQL MCP — your AI writes SQL, we validate and run it.

ChatGPT custom app

Settings → Apps → Create app → Server URL: https://querygate.vercel.app/sse
Paste your Postgres URL in chat once — the AI calls connect and gets an access_token. Use that JWT on all later requests (no raw URL in headers).

After connect — optional header

{
  "mcpServers": {
    "querygate": {
      "url": "https://querygate.vercel.app/sse",
      "headers": {
        "Authorization": "Bearer <access_token from connect>"
      }
    }
  }
}

Cursor / Claude remote (Streamable HTTP): https://querygate.vercel.app/mcp

Ready-to-copy JSON: /setup

Local stdio (Cursor, Claude Desktop)

{
  "mcpServers": {
    "querygate": {
      "command": "node",
      "args": ["/path/to/querygate/dist/cli.js", "--stdio"],
      "env": {
        "DATABASE_URL": "postgres://user:password@host:5432/mydb"
      }
    }
  }
}

Or omit DATABASE_URL from env — user pastes their connection string in chat, then the AI calls connect with database_url.
