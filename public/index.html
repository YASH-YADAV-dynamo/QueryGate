QueryGate

Read-only PostgreSQL MCP — your AI writes SQL, we validate and run it.

ChatGPT custom app

Settings → Apps → Create app → Server URL:
Add header DATABASE_URL with your Postgres connection string.

Local stdio (Cursor, Claude Desktop)

{
  "mcpServers": {
    "querygate": {
      "command": "node",
      "args": ["/path/to/querygate/dist/cli.js", "--stdio"],
      "env": {
        "DATABASE_URL": "postgres://user:password@host:5432/mydb"
      }
    }
  }
}

Remote MCP (HTTPS)

Use this fixed URL in any remote MCP client:

/mcp

Client config

Same JSON for every app — replace DATABASE_URL with your Postgres connection string:

{
  "mcpServers": {
    "querygate": {
      "url": "https://YOUR-DOMAIN/mcp",
      "headers": {
        "DATABASE_URL": "postgres://user:password@host:5432/mydb"
      }
    }
  }
}
