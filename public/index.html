QueryGate

Read-only PostgreSQL MCP — your AI writes SQL, we validate and run it.

MCP endpoint

Use this URL in Cursor, Claude, or ChatGPT:

https://querygate.vercel.app/mcp

Client config

Replace DATABASE_URL with your Postgres connection string:

{
  "mcpServers": {
    "querygate": {
      "url": "https://querygate.vercel.app/mcp",
      "headers": {
        "DATABASE_URL": "postgres://user:password@host:5432/mydb"
      }
    }
  }
}

Ready-to-copy JSON: /setup

ChatGPT custom app

Settings → Apps → Create app → Server URL: https://querygate.vercel.app/mcp
Add header DATABASE_URL with your Postgres connection string.

Local stdio (Cursor, Claude Desktop)

{
  "mcpServers": {
    "querygate": {
      "command": "node",
      "args": ["/path/to/querygate/dist/cli.js", "--stdio"],
      "env": {
        "DATABASE_URL": "postgres://user:password@host:5432/mydb"
      }
    }
  }
}
